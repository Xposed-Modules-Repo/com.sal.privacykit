# Privacy Kit

Privacy Kit provides per-app Android identifier spoofing for LSPosed.

This public repository is used for LSPosed module distribution and release
metadata. It does not publish the private source code or build project.

## Module

- Package: `com.sal.privacykit`
- Framework: LSPosed / Xposed
- Current release: `5-1.4`
- APK: [app-release.apk](https://github.com/Xposed-Modules-Repo/com.sal.privacykit/releases/download/5-1.4/app-release.apk)

## Installation

Install the APK from the GitHub release or through the LSPosed module
repository, then enable the module in LSPosed for the apps you want to manage.

## Disclaimer

Use this module only on devices and apps where you understand and accept the
privacy, compatibility, and policy implications. Some apps may restrict access
or behave unexpectedly when device identifiers are changed.
